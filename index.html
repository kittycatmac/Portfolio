Alley art

Our group built a full stack street smart art app. It allows street artists to create porfiles, upload pictures, and opt to communicate with the public. On the other side, users can search for art by either location or artist. Once the user has located art pieces, they can map locations, view the artist's profile or save art to favorites. So, look up art and go on an adventure

REACT application with Redux managing user states, MongoDB was used for the database, Javascript for controllers and CSS and Materialize for front end UI/UX. Additional technologies: Google Maps, Google Geocoder, AWS S3 Bucket (photos), Passport (local strategy)

Features to add: Ineractive google map to zoom into street views, favorite pins, and eventually instant messaging.
